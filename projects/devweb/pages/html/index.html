HTML

Estrutura básica do HTML

Todo documento HTML começa com uma declaração que define o tipo de documento e é seguida pela tag <html> que envolve todo o conteúdo da página. O <html> é então dividido em duas seções: <head> e <body>. A seção <head> contém informações sobre a página, como o título e os metadados, enquanto a seção <body> contém todo o conteúdo que será exibido na página. Aqui está um exemplo de estrutura básica do HTML:

<!DOCTYPE html>
<html>
<head>
<title>Minha primeira página HTML</title>
</head>
<body>
<h1>Bem-vindo ao meu site!</h1>
<p>Este é um exemplo de um parágrafo HTML.</p>
</body>
</html>

Tags HTML

As tags HTML são os blocos de construção do HTML e são usadas para definir o conteúdo e a estrutura da página. As tags são escritas entre colchetes angulares <> e são geralmente usadas em pares, com uma tag de abertura <tag> e uma tag de fechamento </tag>. Algumas tags HTML comuns são:

<h1> a <h6>: tags de cabeçalho que definem títulos e subtítulos de diferentes níveis.
<p>: tag de parágrafo que define um bloco de texto.
<a>: tag de âncora que cria um link para outra página ou recurso.
<img>: tag de imagem que exibe uma imagem na página.
<ul> e <li>: tags de lista que criam uma lista não ordenada.
<ol> e <li>: tags de lista que criam uma lista ordenada.
<div>: tag de divisão que define um contêiner para outros elementos HTML.
<span>: tag de span que define uma seção de texto dentro de um bloco de texto.

Atributos HTML

Os atributos HTML são usados para fornecer informações adicionais sobre as tags HTML. Eles são escritos dentro das tags e geralmente possuem um nome e um valor. Alguns atributos comuns são:

href: usado na tag <a> para especificar o URL da página ou recurso vinculado. src: usado na tag <img> para especificar o URL da imagem a ser exibida. alt: usado na tag <img> para especificar o texto alternativo para a imagem, que é exibido caso a imagem não possa ser carregada. class e id: usados em várias tags HTML para identificar e estilizar elementos específicos da página.

CSS e JavaScript

O HTML é apenas uma parte da criação de uma página web. Para estilizar e adicionar interatividade à página, você também precisará de CSS (Cascading Style Sheets) e JavaScript. O CSS é usado para estilizar o layout e o design da página, enquanto o JavaScript é usado para adicionar interatividade e funcionalidade à página.
